# 🛡️ V.K. Transmission Console

A secure, sleek Streamlit application designed for transmitting encrypted agent messages to headquarters. 

Created and secured by **LORD VAMSI KRISHNA ⚡**

## 🚀 Features

- **Agent Codename Authentication**: Restricts console access to authorized agents.
- **Encrypted Transmission Gate**: Validates inputs to prevent empty command broadcasts.
- **Vision's Power Scan (Token Cost Estimator)**: Dynamically estimates the mainframe token compute cost (1 token ≈ 4 characters) before beam-up.

---

## 🛠️ Installation & Setup

Ensure you have Python installed, then follow these steps:

1. **Install Streamlit**:
   ```bash
   pip install streamlit
   ```

2. **Run the Application**:
   ```bash
   streamlit run app.py
   ```

---

## 🖥️ Usage

1. Enter your agent codename.
2. Enter your encrypted message.
3. Click the **🚀 Transmit** button to uplink the message to headquarters.
